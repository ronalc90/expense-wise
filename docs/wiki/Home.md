# ExpenseWise Wiki

Bienvenido a la wiki de ExpenseWise, una API REST para control inteligente de gastos dirigida a freelancers y profesionales independientes.

## Resumen del Proyecto

- **Stack:** Java 21 + Spring Boot 3.2 + PostgreSQL 16
- **Autenticacion:** JWT con HMAC-SHA256
- **Tests:** 39 tests (27 integracion + 12 unitarios)
- **Exportacion:** CSV y PDF
- **Frontend:** Glassmorphism UI integrado
